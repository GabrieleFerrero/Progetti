# LeapProcessing library

With this library we are going to manage the inputs that the [LeapLeap](https://www.ultraleap.com/product/leap-motion-controller/) gives us back by going to draw them with processing.

## Libraries used

- [Java Arduino Communication Library](https://github.com/HirdayGupta/Java-Arduino-Communication-Library)
- [Leap Motion](https://developer-archive.leapmotion.com/documentation/java/devguide/Leap_SDK_Overview.html)
- [Processing](https://processing.org/)
